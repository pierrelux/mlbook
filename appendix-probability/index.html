Annexe: Révision de probabilités

Cette annexe révise les concepts de probabilité utilisés tout au long du livre.

Variables aléatoires¶

Définitions¶

Une variable aléatoire est une fonction qui associe un résultat d’une expérience aléatoire à un nombre réel.

  • Variable discrète: prend un nombre fini ou dénombrable de valeurs

  • Variable continue: prend des valeurs dans un intervalle de R\mathbb{R}R

Fonction de masse de probabilité (PMF)¶

Pour une variable discrète XXX, la PMF est:

p(x)=P(X=x)p(x) = P(X = x)p(x)=P(X=x)
(1)

Propriétés:

  • p(x)≥0p(x) \geq 0p(x)≥0 pour tout xxx

  • ∑xp(x)=1\sum_x p(x) = 1∑x​p(x)=1

Fonction de densité de probabilité (PDF)¶

Pour une variable continue XXX, la PDF f(x)f(x)f(x) satisfait:

P(a≤X≤b)=∫abf(x) dxP(a \leq X \leq b) = \int_a^b f(x) \, dxP(a≤X≤b)=∫ab​f(x)dx
(2)

Propriétés:

  • f(x)≥0f(x) \geq 0f(x)≥0 pour tout xxx

  • ∫−∞∞f(x) dx=1\int_{-\infty}^{\infty} f(x) \, dx = 1∫−∞∞​f(x)dx=1

Note: f(x)f(x)f(x) peut être >1> 1>1; c’est une densité, pas une probabilité.

Distributions courantes¶

Distribution de Bernoulli¶

Variable binaire X∈{0,1}X \in \{0, 1\}X∈{0,1}:

p(x∣θ)=θx(1−θ)1−xp(x \mid \theta) = \theta^x (1-\theta)^{1-x}p(x∣θ)=θx(1−θ)1−x
(3)

  • Paramètre: θ∈[0,1]\theta \in [0, 1]θ∈[0,1] (probabilité de succès)

  • Espérance: E[X]=θ\mathbb{E}[X] = \thetaE[X]=θ

  • Variance: Var(X)=θ(1−θ)\text{Var}(X) = \theta(1-\theta)Var(X)=θ(1−θ)

Distribution catégorique (multinoulli)¶

Variable à KKK catégories X∈{1,…,K}X \in \{1, \ldots, K\}X∈{1,…,K}:

p(x=k∣θ)=θkp(x = k \mid \boldsymbol{\theta}) = \theta_kp(x=k∣θ)=θk​

où θ=(θ1,…,θK)\boldsymbol{\theta} = (\theta_1, \ldots, \theta_K)θ=(θ1​,…,θK​) avec ∑kθk=1\sum_k \theta_k = 1∑k​θk​=1.

Avec le codage one-hot x∈{0,1}K\boldsymbol{x} \in \{0,1\}^Kx∈{0,1}K:

p(x∣θ)=∏k=1Kθkxkp(\boldsymbol{x} \mid \boldsymbol{\theta}) = \prod_{k=1}^K \theta_k^{x_k}p(x∣θ)=k=1∏K​θkxk​​
(5)

Distribution binomiale¶

Nombre de succès en nnn essais de Bernoulli:

p(k∣n,θ)=(nk)θk(1−θ)n−kp(k \mid n, \theta) = \binom{n}{k} \theta^k (1-\theta)^{n-k}p(k∣n,θ)=(kn​)θk(1−θ)n−k

  • Espérance: E[X]=nθ\mathbb{E}[X] = n\thetaE[X]=nθ

  • Variance: Var(X)=nθ(1−θ)\text{Var}(X) = n\theta(1-\theta)Var(X)=nθ(1−θ)

Distribution de Poisson¶

Nombre d’événements dans un intervalle:

p(k∣λ)=λke−λk!p(k \mid \lambda) = \frac{\lambda^k e^{-\lambda}}{k!}p(k∣λ)=k!λke−λ​

  • Espérance: E[X]=λ\mathbb{E}[X] = \lambdaE[X]=λ

  • Variance: Var(X)=λ\text{Var}(X) = \lambdaVar(X)=λ

Distribution gaussienne (normale)¶

Univariée:

N(x∣μ,σ2)=12πσ2exp⁡(−(x−μ)22σ2)\mathcal{N}(x \mid \mu, \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)N(x∣μ,σ2)=2πσ2​1​exp(−2σ2(x−μ)2​)

  • Espérance: E[X]=μ\mathbb{E}[X] = \muE[X]=μ

  • Variance: Var(X)=σ2\text{Var}(X) = \sigma^2Var(X)=σ2

Multivariée:

N(x∣μ,Σ)=1(2π)D/2∣Σ∣1/2exp⁡(−12(x−μ)⊤Σ−1(x−μ))\mathcal{N}(\boldsymbol{x} \mid \boldsymbol{\mu}, \boldsymbol{\Sigma}) = \frac{1}{(2\pi)^{D/2}|\boldsymbol{\Sigma}|^{1/2}} \exp\left(-\frac{1}{2}(\boldsymbol{x}-\boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1}(\boldsymbol{x}-\boldsymbol{\mu})\right)N(x∣μ,Σ)=(2π)D/2∣Σ∣1/21​exp(−21​(x−μ)⊤Σ−1(x−μ))

où:

  • μ∈RD\boldsymbol{\mu} \in \mathbb{R}^Dμ∈RD est le vecteur moyenne

  • Σ∈RD×D\boldsymbol{\Sigma} \in \mathbb{R}^{D \times D}Σ∈RD×D est la matrice de covariance (symétrique, définie positive)

Distribution uniforme¶

Continue sur [a,b][a, b][a,b]:

p(x)=1b−aI(a≤x≤b)p(x) = \frac{1}{b-a} \mathbb{I}(a \leq x \leq b)p(x)=b−a1​I(a≤x≤b)

Distribution exponentielle¶

p(x∣λ)=λe−λxI(x≥0)p(x \mid \lambda) = \lambda e^{-\lambda x} \mathbb{I}(x \geq 0)p(x∣λ)=λe−λxI(x≥0)

Distribution Beta¶

Pour x∈[0,1]x \in [0, 1]x∈[0,1]:

Beta(x∣α,β)=xα−1(1−x)β−1B(α,β)\text{Beta}(x \mid \alpha, \beta) = \frac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha, \beta)}Beta(x∣α,β)=B(α,β)xα−1(1−x)β−1​

où B(α,β)=Γ(α)Γ(β)Γ(α+β)B(\alpha, \beta) = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}B(α,β)=Γ(α+β)Γ(α)Γ(β)​.

Utile comme a priori pour les paramètres de Bernoulli (conjugué).

Généralisation de Beta à KKK dimensions:

Dir(θ∣α)=1B(α)∏k=1Kθkαk−1\text{Dir}(\boldsymbol{\theta} \mid \boldsymbol{\alpha}) = \frac{1}{B(\boldsymbol{\alpha})} \prod_{k=1}^K \theta_k^{\alpha_k - 1}Dir(θ∣α)=B(α)1​k=1∏K​θkαk​−1​

Espérance et variance¶

Espérance¶

L’espérance (ou moyenne) d’une variable aléatoire:

Discrète: E[X]=∑xx p(x)\mathbb{E}[X] = \sum_x x \, p(x)E[X]=∑x​xp(x)

Continue: E[X]=∫x f(x) dx\mathbb{E}[X] = \int x \, f(x) \, dxE[X]=∫xf(x)dx

Propriétés:

  • Linéarité: E[aX+bY]=aE[X]+bE[Y]\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y]E[aX+bY]=aE[X]+bE[Y]

  • E[g(X)]=∑xg(x)p(x)\mathbb{E}[g(X)] = \sum_x g(x) p(x)E[g(X)]=∑x​g(x)p(x) (ou intégrale pour le continu)

Variance¶

La variance mesure la dispersion autour de la moyenne:

Var(X)=E[(X−E[X])2]=E[X2]−E[X]2\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - \mathbb{E}[X]^2Var(X)=E[(X−E[X])2]=E[X2]−E[X]2
(14)

L’écart-type: σ=Var(X)\sigma = \sqrt{\text{Var}(X)}σ=Var(X)​

Propriétés:

  • Var(aX+b)=a2Var(X)\text{Var}(aX + b) = a^2 \text{Var}(X)Var(aX+b)=a2Var(X)

  • Si X⊥YX \perp YX⊥Y: Var(X+Y)=Var(X)+Var(Y)\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y)Var(X+Y)=Var(X)+Var(Y)

Covariance¶

Cov(X,Y)=E[(X−E[X])(Y−E[Y])]=E[XY]−E[X]E[Y]\text{Cov}(X, Y) = \mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])] = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y]Cov(X,Y)=E[(X−E[X])(Y−E[Y])]=E[XY]−E[X]E[Y]
(15)

Corrélation (covariance normalisée):

ρ(X,Y)=Cov(X,Y)σXσY∈[−1,1]\rho(X, Y) = \frac{\text{Cov}(X, Y)}{\sigma_X \sigma_Y} \in [-1, 1]ρ(X,Y)=σX​σY​Cov(X,Y)​∈[−1,1]
(16)

Matrice de covariance pour un vecteur aléatoire X\boldsymbol{X}X:

Σ=E[(X−μ)(X−μ)⊤]\boldsymbol{\Sigma} = \mathbb{E}[(\boldsymbol{X} - \boldsymbol{\mu})(\boldsymbol{X} - \boldsymbol{\mu})^\top]Σ=E[(X−μ)(X−μ)⊤]
(17)

Probabilités jointes et marginales¶

Règle de la somme (marginalisation)¶

p(X)=∑Yp(X,Y)oup(x)=∫p(x,y) dyp(X) = \sum_Y p(X, Y) \quad \text{ou} \quad p(x) = \int p(x, y) \, dyp(X)=Y∑​p(X,Y)oup(x)=∫p(x,y)dy
(18)

Règle du produit¶

p(X,Y)=p(X∣Y)p(Y)=p(Y∣X)p(X)p(X, Y) = p(X \mid Y) p(Y) = p(Y \mid X) p(X)p(X,Y)=p(X∣Y)p(Y)=p(Y∣X)p(X)
(19)

Règle de la chaîne¶

p(X1,…,Xn)=∏i=1np(Xi∣X1,…,Xi−1)p(X_1, \ldots, X_n) = \prod_{i=1}^n p(X_i \mid X_1, \ldots, X_{i-1})p(X1​,…,Xn​)=i=1∏n​p(Xi​∣X1​,…,Xi−1​)
(20)

Théorème de Bayes¶

Formulation¶

p(θ∣D)=p(D∣θ)p(θ)p(D)p(\theta \mid \mathcal{D}) = \frac{p(\mathcal{D} \mid \theta) p(\theta)}{p(\mathcal{D})}p(θ∣D)=p(D)p(D∣θ)p(θ)​
(21)

où:

  • p(θ)p(\theta)p(θ): a priori (croyance avant les données)

  • p(D∣θ)p(\mathcal{D} \mid \theta)p(D∣θ): vraisemblance (probabilité des données étant donné θ\thetaθ)

  • p(θ∣D)p(\theta \mid \mathcal{D})p(θ∣D): a posteriori (croyance après avoir vu les données)

  • p(D)p(\mathcal{D})p(D): évidence ou constante de normalisation

Constante de normalisation¶

p(D)=∫p(D∣θ)p(θ) dθp(\mathcal{D}) = \int p(\mathcal{D} \mid \theta) p(\theta) \, d\thetap(D)=∫p(D∣θ)p(θ)dθ
(22)

Cette intégrale est souvent difficile à calculer (inférence approximative).

Forme proportionnelle¶

Souvent, on écrit:

p(θ∣D)∝p(D∣θ)p(θ)p(\theta \mid \mathcal{D}) \propto p(\mathcal{D} \mid \theta) p(\theta)p(θ∣D)∝p(D∣θ)p(θ)
(23)

(a posteriori proportionnel à vraisemblance × a priori)

Indépendance¶

Indépendance marginale¶

X⊥YX \perp YX⊥Y si:

p(X,Y)=p(X)p(Y)p(X, Y) = p(X) p(Y)p(X,Y)=p(X)p(Y)
(24)

Équivalent à: p(X∣Y)=p(X)p(X \mid Y) = p(X)p(X∣Y)=p(X)

Indépendance conditionnelle¶

X⊥Y∣ZX \perp Y \mid ZX⊥Y∣Z si:

p(X,Y∣Z)=p(X∣Z)p(Y∣Z)p(X, Y \mid Z) = p(X \mid Z) p(Y \mid Z)p(X,Y∣Z)=p(X∣Z)p(Y∣Z)
(25)

Attention: X⊥YX \perp YX⊥Y n’implique pas X⊥Y∣ZX \perp Y \mid ZX⊥Y∣Z (et vice versa).

Divergence de Kullback-Leibler¶

Définition¶

La divergence KL mesure la “distance” entre deux distributions:

DKL(p∥q)=Ep[log⁡p(x)q(x)]=∑xp(x)log⁡p(x)q(x)D_{\text{KL}}(p \| q) = \mathbb{E}_p\left[\log \frac{p(x)}{q(x)}\right] = \sum_x p(x) \log \frac{p(x)}{q(x)}DKL​(p∥q)=Ep​[logq(x)p(x)​]=x∑​p(x)logq(x)p(x)​
(26)

Propriétés¶

  • DKL(p∥q)≥0D_{\text{KL}}(p \| q) \geq 0DKL​(p∥q)≥0 (inégalité de Gibbs)

  • DKL(p∥q)=0D_{\text{KL}}(p \| q) = 0DKL​(p∥q)=0 ssi p=qp = qp=q

  • Non symétrique: DKL(p∥q)≠DKL(q∥p)D_{\text{KL}}(p \| q) \neq D_{\text{KL}}(q \| p)DKL​(p∥q)=DKL​(q∥p) en général

  • Ce n’est pas une distance (ne satisfait pas l’inégalité triangulaire)

Pour les gaussiennes¶

Pour p=N(μ1,Σ1)p = \mathcal{N}(\boldsymbol{\mu}_1, \boldsymbol{\Sigma}_1)p=N(μ1​,Σ1​) et q=N(μ2,Σ2)q = \mathcal{N}(\boldsymbol{\mu}_2, \boldsymbol{\Sigma}_2)q=N(μ2​,Σ2​):

DKL(p∥q)=12[log⁡∣Σ2∣∣Σ1∣−D+tr(Σ2−1Σ1)+(μ2−μ1)⊤Σ2−1(μ2−μ1)]D_{\text{KL}}(p \| q) = \frac{1}{2}\left[\log\frac{|\boldsymbol{\Sigma}_2|}{|\boldsymbol{\Sigma}_1|} - D + \text{tr}(\boldsymbol{\Sigma}_2^{-1}\boldsymbol{\Sigma}_1) + (\boldsymbol{\mu}_2 - \boldsymbol{\mu}_1)^\top \boldsymbol{\Sigma}_2^{-1}(\boldsymbol{\mu}_2 - \boldsymbol{\mu}_1)\right]DKL​(p∥q)=21​[log∣Σ1​∣∣Σ2​∣​−D+tr(Σ2−1​Σ1​)+(μ2​−μ1​)⊤Σ2−1​(μ2​−μ1​)]
(27)

Entropie¶

Entropie de Shannon¶

L’entropie mesure l’incertitude d’une distribution:

H(X)=−E[log⁡p(X)]=−∑xp(x)log⁡p(x)\mathbb{H}(X) = -\mathbb{E}[\log p(X)] = -\sum_x p(x) \log p(x)H(X)=−E[logp(X)]=−x∑​p(x)logp(x)
(28)

  • Maximale pour une distribution uniforme

  • Minimale (= 0) pour une distribution déterministe

Entropie croisée¶

H(p,q)=−Ep[log⁡q(X)]=−∑xp(x)log⁡q(x)\mathbb{H}(p, q) = -\mathbb{E}_p[\log q(X)] = -\sum_x p(x) \log q(x)H(p,q)=−Ep​[logq(X)]=−x∑​p(x)logq(x)
(29)

Relation avec KL:

H(p,q)=H(p)+DKL(p∥q)\mathbb{H}(p, q) = \mathbb{H}(p) + D_{\text{KL}}(p \| q)H(p,q)=H(p)+DKL​(p∥q)
(30)
